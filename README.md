Canvas API Tracker

A full-stack web application that connects to the Canvas LMS API to track assignments, deadlines, and course progress within a customizable time range. Built to give a clear, real-time view of what’s due and how you’re doing across all your courses.

Overview

This project was created to simplify how students interact with their Canvas data. Instead of manually checking each course, the app pulls everything into one place and displays upcoming assignments along with progress insights. It focuses on fast data retrieval, clean UI, and secure API handling.
